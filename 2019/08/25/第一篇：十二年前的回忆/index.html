第一篇：十二年前的回忆——淫魔妖女（一）

发表于 2019-08-25 更新于 2019-08-27 分类于 里番 阅读次数： 评论数：
本文字数： 3.1k 阅读时长 ≈ 3 分钟
最近各大视频网站到处可以看到，我冲了，手冲之类的评论。让本绅士倍感怀念，想起了自己那荷尔蒙爆炸的年头。
第一次听说手冲这个词大约是在12-13年前,当时本茄还是一个年少无知而又气血旺盛的骚年。
当时和基友们讨论施法材料的时候，无意间听到有人传出了“手冲”这个富有生气的代名词。
“手冲”这个词能火，妙就妙在它让人联想到很多有活力的画面。
比如：手握冲击钻的建筑工人👷，亦或是一个手握冲锋枪的壮汉💪。
更直接的是能让人想到自己用手做冲程这个动作。🚕
从此之后，我们便用手冲代替了打飞机来称呼大家的“日常娱乐”。
那么手冲的素材从哪里来呢？
依靠网上下载施法材料来施法？当时的网络速度不敢恭维，用电驴或者种子下一个AV可能就要1-2天的时间。关键是资源还少，并且缺少相关的各种资料。除非是十分知名的作品，不然自己不知道下的是谁，作品叫什么名字。
那么年轻人旺盛的荷尔蒙要通过何种渠道来宣泄一下呢？
这种时候要夸奖一下万能的盗版碟💽商贩了。当时贫乏的娱乐完全依靠他们的碟片来支撑。
各类里番的剪辑版、繁体字幕版或者是粤语版。依靠着各种刻录盘和VCD，传播于各大淫民群众之中。
有一天，一个基友告诉我有个好东西要给我康康。当天下午给我一张没有封面的刻录碟，里面是几部里番的剪辑版。令人震惊的是这玩意竟然还做了反盗版处理，真的是应了那句冯巩的名言：越是贼窝越要加强防盗工作。（以后我们可以可以谈谈里面其他几部，也很经典。）
我和这一部淫魔妖女就这么邂逅了，并开启了我的里番鉴赏时代。
这张碟片的内容只有一小段，而这一小段成为了我印象最为深刻的一幕。



以上这三张图为这一段的精彩截图。而这一段正好就是整部片里最为细致的部分。里面女主的恐惧，未知BOSS的残暴，还有丰富的物理引擎（乳摇），简直是让人无妨抗拒。我都忘记我这一段用了多少次了。
对本绅士而言，老片有老片的好。现在新制作的里番，乳摇越来越假，人物越来越崩，镜头越来越晃，都快让人吐了。还怎么让人产生手冲的冲动？
老片在很多互动上的细腻动作，是现在新作品表现不出来的。而这些细节让我感到欣赏这些老番就像在美术馆欣赏艺术品一样。
怀念就到这里，我们接下来来谈一下这部片的基本素质吧。
《淫魔妖女》这部作品发售于1994年，一共有5个独立的小故事。基本上都是讲述独立女性被置于各个时空被各种OOXX最后获得自我拯救（？）的事件。现在想想其实这部作品剧情非常平权，并不是那种单纯以侮辱女性为主题的手冲作品。里面的元素非常丰富，赛博朋克、中世纪、日本战国、现代、魔法、骑士、机器人等等。
单独拿出来做一部TV都绰绰有余，而它将这些融合在了5集动画之中。说明导演与编剧的功力了得，不愧是里届斯皮尔伯格——乱交太郎企划的作品。
那第一集《MAYA》讲的是什么和其中的冲点在哪，就让我们一起一探究竟吧！


第一幕就是男主被一群人用激光枪追杀，脸还被打出一个伤疤。



接下来便是男主回忆着与初恋女友在村子里的日子，家里有房又有田，生活乐无边。谁知某一天村民突然性情大变，男主被村民架着看自己女友被一群村民轮流肏。不由得想对男主说一句：虽然你女友被肏这段很不实用，但我还是很同情你。要坚强。






接下来女主流泪远去，伴随着男主无助的目光。画面切回男主逃跑，男主突然来了一句：都是因为那个女人！

喵喵喵？这女主提着行李箱怎么看都是才路过啊，流泪明显表现的是对这悲剧的同情啊。男主你咋就知道这个弱不禁风的女主就是罪魁祸首啊？是不是因为村民只肏你女友你心理不平衡，所以旁边路人不被肏就是重点怀疑对象？你这样是不是有一种钦定的感觉？（男主：我不要你觉得，我要我觉得！本茄：好吧。你是主角，你说了算。）

接下来画风一转，镜头找到女主。




一个龙套男拉着女主狂奔逃避直升机追捕。直升机打了几下龙套男就死了，女主无伤。女主面无表情的看着被打死的龙套男。龙套男真是太惨了，死个小猫小狗都比他受关注度高吧。（可能当时的监督就是为了告诉我们舔狗不得好死吧😄）




后来直升机也不明原因的原地爆炸，女主就逃到了沙漠之中。而男主在远处尾行，观察女主的一举一动。



女主到达一座有巨大高塔的城市，里面的市民看上去都非常“和善”。眼睛都冒出红光了，估计是内火旺盛，需要泄火。




女主在休息时候发现一只小猫，就在这时需要泄火的“和善”市民也到了，纷纷要求女主帮助泄火。而男主带着猫在一旁强势OB。（好吧，这可能是男主的特殊癖好：看人泄火。）女主的脸也迎来了一次崩坏，我一下子都没认出这是谁。



女主醒来就发现自己在陌生的房间，管家直接就通知她你已经是鸡🐔了。（这么钦定的吗？）男主于是迫不及待的点名成为第一位客人。看到这里，本绅士不禁发问：男主你的深仇大恨呢？说好的都是因为这个女人，我还以为你要做什么大事，结果还不是来泄火？男主一边泄火一边和女主讨论塔的问题，原来是借用女主要调查城市和塔，还算你有点脑子。而完事后的女主还一脸傲娇的向管家表示泄火没意思。那你前面那么陶醉干什么？本绅士真是对这两货服了。😓




女主会见BOSS，于是发生了文章开头的这一幕。BOSS表示非常满意！点了个赞👍



而男主发现了这个城市的地下制药工厂，顺手牵羊的时候被管家发现。男主腹黑的把管家和BOSS搞♂基的事情在大家面前大公开，表示画面不堪入目，管家脸色大变表示一定要干死男主。（男主看来你的兴趣真的是OB人家泄火。😎）





逃出管家追击的男主来到BOSS和女主的泄火现场，转动阀门威胁BOSS，如果不交出女主就让他体液倒流而死。（身为BOSS留一个这么大的命门，你真的不是故意的吗？）BOSS放开女主之后正好管家赶到，形成三足鼎立之势。管家表示女主（金坷垃）是我哒！男主一记“麻麻的”直拳击倒管家，抱着女主逃了。（BOSS：女主好处都有啥，谁说对了，我也不给他）

emmmmm…话说管家你不是个基佬吗？咋也对女人感兴趣了？看来女主果然有点东西。




后面搞笑的来了，男主开车带女主逃到沙漠以后，直接就抱住女主要当场泄火。结果才没多久管家就开车追上来，把男主一枪🔫干趴。

本绅士寻思着男主你们这是逃命，又不是旅游，咋就突然忘我的野战起来了?





管家表示没有人能阻止我啦，我要带着MAYA走。没想到BOSS追了上来，用自己的八爪鱼🦑把管家捅死了。现场3个男人全部阵亡，女主露出了神秘的微笑。（这样看来女主果然是罪魁祸首，男主我错怪你了。）





男主在弥留之际道出了真相，和这个女人搭上关系人生就完蛋了。虽然和她有深仇大恨，但是我还是控制不住我自己啊！男人这种生物真是可悲啊。（说到底不是你自己小头控制大头，说好的报仇呢？）城市毁灭了，MAYA也消失在沙漠之中。
虽然剧情有这样那样的吐槽点，但总的来说剧情相对连贯，也没有什么非常不合理的硬伤。画面上也基本上保持的比较好，没有出现非常多的崩坏。最重要的手冲度上，女主的几个泄火剧情也非常实用。是一部非常不错的作品。
要说美中不足的地方就是这部片的“逼”真度和泄火时长了。虽然这部有无码版本，但是并没有画B，所有的镜头都是涂黑处理让人很不过瘾。最后就是相对于40多分钟的片长，泄火镜头零散穿插在剧情中，且总共只有12分钟左右，实在让人扫兴。如果冲了一半就来剧情了会影响自己手感，得经常使用播放器的A-B重复功能才能让自己冲的尽兴。

那我们现在来总结一下这部番的评分吧。（单项满分5分）

剧情：⭐⭐⭐⭐

画面：⭐⭐⭐⭐

手冲度：⭐⭐⭐⭐⭐

本片可冲时间：12分钟左右

“逼”真度：无（因为没画🤣）

综合评分：⭐⭐⭐⭐（是一部可以反复冲的好作品）

  40. 40. 男主在弥留之际道出了真相，和这个女人搭上关系人生就完蛋了。虽然和她有深仇大恨，但是我还是控制不住我自己啊！男人这种生物真是可悲啊。（说到底不是你自己小头控制大头，说好的报仇呢？）城市毁灭了，MAYA也消失在沙漠之中。
  41. 41. 虽然剧情有这样那样的吐槽点，但总的来说剧情相对连贯，也没有什么非常不合理的硬伤。画面上也基本上保持的比较好，没有出现非常多的崩坏。最重要的手冲度上，女主的几个泄火剧情也非常实用。是一部非常不错的作品。
  42. 42. 要说美中不足的地方就是这部片的“逼”真度和泄火时长了。虽然这部有无码版本，但是并没有画B，所有的镜头都是涂黑处理让人很不过瘾。最后就是相对于40多分钟的片长，泄火镜头零散穿插在剧情中，且总共只有12分钟左右，实在让人扫兴。如果冲了一半就来剧情了会影响自己手感，得经常使用播放器的A-B重复功能才能让自己冲的尽兴。
